Priyank

M.Tech

Side-channel attacks are a particular class of attacks, usually targeting cryptographic algorithms, which do not exploit a flaw in the design of the algorithms themselves but rather in their implementation. An attacker observes a victim’s pattern of memory access and tries to infer secret information through cache using timing differences between a cache hit and a cache miss. The attacker doesn’t modify the victim’s code and neither do they interact directly with the victim program during runtime. This behaviour makes these attacks stealthy to available defences like an antivirus or an Intrusion detection system. These attacks can be mounted across VMs and without elevated privileges, thus making them very powerful.

My research aims to provide a practical and robust software-based solution for detection of cache based side channel attacks.
